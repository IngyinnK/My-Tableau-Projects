# 📊 Tableau Data Visualisation Project

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on building **interactive dashboards in Tableau** to analyse datasets and communicate insights effectively.

The project demonstrates key **data visualisation and analytics skills**, transforming raw data into meaningful insights through clear and interactive dashboards.

The dashboards explore trends, patterns, and relationships in datasets using multiple visualisation techniques and interactive features.

## 🎯 Project Objective

The goal of this project was to build practical experience in:

- Data visualisation
- Dashboard creation
- Interactive analytics
- Communicating insights through data

These skills are essential for turning complex datasets into **clear, actionable insights for stakeholders**.

## 🛠️ Skills Demonstrated

- 📊 **Interactive Dashboard Design**

- 🎛️ **Filters & Slicers for user interaction**
- 🧮 **Calculated Fields**
- 📖 **Data Storytelling**
- 📈 **Trend Analysis**
- 🎨 **Visual Design Best Practices**

## 📉 Chart Types Used

The dashboards include a variety of visualisations to present data effectively:

| Chart Type | Purpose |
|-------------|---------|
| 📊 **Bar Charts** | Compare categories and performance |
| 📈 **Line Charts** | Show trends over time |
| 🥧 **Pie Charts** | Display proportional distributions |
| 🔵 **Scatter Plots** | Explore relationships between variables |
| 🗺️ **Maps** | Visualise geographic patterns |

## 🎛️ Dashboard Features

### 🔍 Interactive Filters
Users can explore the data dynamically by selecting different categories, regions, or time periods.

### 🧮 Calculated Fields
Custom calculations were created to derive new insights such as:
- averages
- percentages
- comparative metrics

### 🧩 Multi-Chart Dashboards
Multiple visualisations are combined into a single dashboard to create a **comprehensive analytical view**.
